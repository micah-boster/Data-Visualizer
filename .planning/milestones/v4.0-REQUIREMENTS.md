## Out of Scope

| Feature | Reason |
|---------|--------|
| New API routes for chart data | Charting operates on client-side dataset |
| Chart data export (PNG/SVG) | Nice-to-have, not core workflow |
| Real-time data streaming | Batch refresh sufficient for 2-3 users |
| ML-based chart recommendations | Overkill for small dataset |
| Mobile-optimized UI | Desktop-first for internal users |
| User authentication | Small team, defer to v5 |
| Framer Motion or heavy animation lib | CSS transitions + Web Animations API sufficient |

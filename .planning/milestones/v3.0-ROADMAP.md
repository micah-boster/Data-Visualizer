**Status:** SHIPPED 2026-04-14

**Total Plans:** 9

## Overview

The tool tells you what to look at, lets you ask why, and benchmarks partners against each other. Three pillars: deterministic anomaly detection with z-scores, Claude natural language query layer for narrative answers, and cross-partner comparison with percentile rankings and trajectory overlays.

## Milestone Summary

**Key Decisions:**
- Anomaly detection is deterministic (z-scores), not ML-based — per explainability constraint
- Claude narrates data, does not detect anomalies — separation of concerns
- Search bar pattern (not chat) for NLQ — point queries, not conversations
- AI SDK useChat + streamText handle all streaming plumbing — no custom SSE
- Cross-partner uses percentile rank via simple-statistics quantileRank
- Best-in-class partner determined by highest perDollarPlacedRate
- Dollar-weighted trajectory as default (matching business use case)
- Comparison matrix extracted shared types to matrix-types.ts for DRY code

# Requirements: v6.0 Proactive Intelligence & Action

**Defined:** 2026-04-16
**Core Value:** The tool pushes intelligence to the team and connects insights to action — briefings, alerts, MBR prep, and downstream workflows happen without manual effort.

## Weekly Partner Highlights (Phase 52 — flagged for review)

- [ ] **BRIEF-01**: Scheduled weekly run generates per-partner summary from latest data
- [ ] **BRIEF-02**: Highlights include: key metric changes, new anomalies, target tracking status, scorecard divergence changes
- [ ] **BRIEF-03**: Delivered as a Notion page organized in a database
- [ ] **BRIEF-04**: Slack notification links to the Notion summary
- [ ] **BRIEF-05**: Highlights reference triangulated data when available

## Pattern Alerts

- [ ] **ALERT-01**: Pattern library: consecutive decline, acceleration, divergence widening, seasonal anomaly, peer-group outlier
- [ ] **ALERT-02**: Alerts delivered to Slack with rich context (mini-chart, comparison data, tool link)
- [ ] **ALERT-03**: Severity tiering: informational, attention-needed, urgent
- [ ] **ALERT-04**: Snooze alerts (1 week, 1 month, forever) with optional note
- [ ] **ALERT-05**: Acknowledge alert and optionally create Notion follow-up task
- [ ] **ALERT-06**: Digest mode: batch low-severity alerts into daily summary
- [ ] **ALERT-07**: Alerts reference triangulated data when available

## MBR Pipeline Integration

- [ ] **MBR-01**: "Prep MBR" button on partner detail view triggers data staging
- [ ] **MBR-02**: Auto-stages KPIs, collection curves, anomalies, target tracking, scorecard comparison
- [ ] **MBR-03**: Staged data feeds into existing MBR pipeline skill with pre-populated context
- [ ] **MBR-04**: Claude generates narrative talking points from triangulated data
- [ ] **MBR-05**: Diff from last MBR: delta values and direction for all tracked metrics
- [ ] **MBR-06**: User can review/edit staged content before triggering deck generation
- [ ] **MBR-07**: MBR prep history: track generated MBRs with timestamps and included data

## Action Connections

- [ ] **ACT-01**: "Flag in Slack" posts rich message with data context, mini-chart, and tool link
- [ ] **ACT-02**: "Create Notion task" creates task in Partnerships Task Tracker with auto-filled context
- [ ] **ACT-03**: "Add to MBR agenda" stages insight for next MBR prep
- [ ] **ACT-04**: Action audit trail logs every action (who, what, when, attached data)
- [ ] **ACT-05**: Actions accessible from right-click context menu on rows and chart data points

## Temporal Intelligence

- [ ] **TEMP-01**: Vintage comparison: same-age batch side-by-side curve overlay
- [ ] **TEMP-02**: Cohort trending: group by vintage quarter, overlay trajectories
- [ ] **TEMP-03**: Forecasting: "if current trends hold, Partner X finishes at $Y" with multiple models
- [ ] **TEMP-04**: Leading indicator surface: early-batch metrics correlated with late-batch outcomes
- [ ] **TEMP-05**: Temporal context in Claude NLQ ("when did this start?", "where is this heading?")
- [ ] **TEMP-06**: All temporal views respect partner list filtering and saved view state

## NLQ Enhancements

- [ ] **NLQ-01**: Follow-up suggestions after each response
- [ ] **NLQ-02**: Clickable data references in responses (drill to source)
- [ ] **NLQ-03**: Query history persistence in Notion
- [ ] **NLQ-04**: Multi-source queries (internal + scorecard + target data)
- [ ] **NLQ-05**: Temporal queries ("when did X start declining?", "will they hit the Q4 target?")

## Future Requirements (deferred to v6.5+)

- **ACT-06**: HubSpot integration: create/update deal from insight
- **ACT-07**: Email draft generation for partner outreach from data context
- **ALERT-08**: Adaptive alert thresholds that learn from snooze/acknowledge patterns
- **BRIEF-06**: Custom briefing templates per partner or per team member
- **TEMP-07**: Seasonal decomposition (STL/Prophet) for time-aware anomaly detection
- **NLQ-06**: Voice input for queries (mobile/tablet use case)

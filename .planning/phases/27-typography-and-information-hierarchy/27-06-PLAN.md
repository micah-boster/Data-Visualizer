**Step A — Create `scripts/check-type-tokens.sh`.**

Create the directory if it does not exist, then write the script. Use these exact contents (tested pattern — uses ripgrep; ripgrep is available):

```bash
#!/usr/bin/env bash
set -euo pipefail

# Phase 27 type-token enforcement.
# Forbids ad-hoc Tailwind text-size and font-weight classes outside an allowlist.
# Allowlist: shadcn primitives + tokens reference page (see docs/TYPE-MIGRATION.md §8).

EXCLUDES=(
  '--glob=!src/components/ui/**'
  '--glob=!src/app/tokens/**'
  '--glob=!src/components/tokens/**'
  '--glob=!*.css'
  '--glob=!*.md'
)

FAIL=0

echo "Checking for ad-hoc text-size classes…"
if rg -n "text-(xs|sm|base|lg|xl|2xl|3xl|4xl)\b" src/ "${EXCLUDES[@]}"; then
  echo ""
  echo "❌ Found ad-hoc text-size classes. Use text-display / text-heading / text-title / text-body / text-label / text-caption." 1>&2
  echo "   See docs/TYPE-MIGRATION.md for the mapping." 1>&2
  FAIL=1
fi

echo ""
echo "Checking for ad-hoc font-weight classes…"
if rg -n "\bfont-(semibold|medium|bold|light|thin|extrabold|black)\b" src/ "${EXCLUDES[@]}"; then
  echo ""
  echo "❌ Found ad-hoc font-weight classes. Type tokens own weight — drop the utility." 1>&2
  echo "   See docs/TYPE-MIGRATION.md §5 for the weight policy." 1>&2
  FAIL=1
fi

if [ "$FAIL" -eq 1 ]; then
  exit 1
fi

echo ""
echo "✅ Type tokens enforced — no ad-hoc classes outside allowlist."
```

After writing, run `chmod +x scripts/check-type-tokens.sh`.

**Note on word-boundaries:**
- `\b` before `font-` prevents matches against e.g. `sans-font-medium` (unlikely but defensive).
- `\b` after `2xl` prevents matches against `text-2xl-foo` (not a real class).
- `text-(xs|sm|base|lg|xl|2xl|3xl|4xl)\b` — the `\b` after the group catches `text-sm` but not `text-small-print` (again, defensive).

2. Create `src/lib/table/aggregations.ts`:
   - `computeAggregates(rows: Row<Record<string, unknown>>[], visibleColumns: Column<Record<string, unknown>>[])`
   - For each visible column, compute based on its `meta.type`:
     - currency/number: sum and average
     - count: sum and count of non-null
     - percentage: average (not sum -- summing percentages is meaningless)
     - text/date: count of non-null values
   - Return: `Record<string, { primary: number | null; label: string }>` where primary is the main aggregate value and label describes it (e.g., "Sum", "Avg", "Count")
   - Use `useMemo`-compatible pure function. The component will memoize.

3. Create `src/components/table/pinning-styles.ts`:
   - Export `getCommonPinningStyles<T>(column: Column<T>): CSSProperties` following the TanStack Table sticky column pinning example from research.
   - Must set explicit background color matching the theme. For pinned cells, use CSS variables: `var(--background)` for light mode base, `var(--card)` for dark mode. The component applying these styles will also need to handle zebra-stripe rows by passing an `isEvenRow` parameter.
   - Actually, make the function signature `getCommonPinningStyles<T>(column: Column<T>, isEvenRow?: boolean): CSSProperties` and include a slightly different background for even rows on pinned columns. Use the theme's muted color for even rows: `var(--muted)`.

4. Create `src/components/table/sort-indicator.tsx`:
   - Small component rendering sort state for a column header.
   - Unsorted: show `ArrowUpDown` icon in muted color (clickable affordance).
   - Ascending: `ArrowUp` icon.
   - Descending: `ArrowDown` icon.
   - Multi-sort: show numbered badge (circled number) next to the arrow when `column.getSortIndex() > 0`. Use small rounded pill with primary bg.
   - Use lucide-react icons. Keep component small and focused.

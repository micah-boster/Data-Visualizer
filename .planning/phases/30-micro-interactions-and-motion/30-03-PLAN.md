```tsx
function Button({
  className,
  variant = "default",
  size = "default",
  ...props
}: ButtonPrimitive.Props & VariantProps<typeof buttonVariants>) {
  const pressScale = variant === "default" || variant === "secondary";
  return (
    <ButtonPrimitive
      data-slot="button"
      data-press-scale={pressScale || undefined}
      className={cn(buttonVariants({ variant, size, className }))}
      {...props}
    />
  )
}
```

Writing `undefined` for non-scaled variants omits the attribute entirely (React skips `undefined` attribute values); scaled variants emit `data-press-scale="true"`.

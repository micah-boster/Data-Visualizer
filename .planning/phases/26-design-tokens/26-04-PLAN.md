**Step A — globals.css density selector block.**

Append to `src/app/globals.css` (inside `@layer base` if one exists, else create one after the existing @theme / :root / .dark blocks):

```css
@layer base {
  [data-density="dense"] {
    --row-height: var(--table-row-height-dense);
    --row-padding-y: var(--spacing-2); /* 8px */
    --row-padding-x: var(--spacing-3); /* 12px */
  }
  [data-density="sparse"] {
    --row-height: var(--table-row-height-sparse);
    --row-padding-y: var(--spacing-3); /* 12px */
    --row-padding-x: var(--spacing-4); /* 16px */
  }
}
```

This means any element with `data-density="dense"` (or "sparse") propagates `--row-height`, `--row-padding-y`, `--row-padding-x` to its descendants. The table container gets `data-density="dense"` by default in Task B.

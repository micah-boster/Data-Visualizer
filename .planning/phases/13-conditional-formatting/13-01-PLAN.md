2. **DeviationResult interface:**
```typescript
export interface DeviationResult {
  zScore: number;
  percentDeviation: number;
  direction: 'above' | 'below' | 'neutral';
  opacity: number;
}
```

3. **computeDeviation(value: number, norm: MetricNorm): DeviationResult | null** — returns null if norm.count < 2 or norm.stddev === 0. Otherwise:
   - Compute zScore = (value - mean) / stddev
   - If abs(zScore) <= 1.5: return direction 'neutral', opacity 0
   - Otherwise: opacity scales linearly from 0 at z=1.5 to maxOpacity (0.35) at z=4, clamped
   - percentDeviation = mean !== 0 ? ((value - mean) / mean) * 100 : 0
   - direction = value > mean ? 'above' : 'below'
   - Polarity: higher is always better, so 'above' = green, 'below' = red

4. **formatDeviationTooltip(formattedValue: string, formattedMean: string, percentDeviation: number): string** — returns tooltip text in the exact user-specified format: "12.3% vs partner avg 18.7% (-34%)" — use sign prefix (+/-).

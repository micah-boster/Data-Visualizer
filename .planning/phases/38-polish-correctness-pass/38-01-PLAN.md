Wrap the Partners SidebarGroup content (app-sidebar.tsx:95-157) with a collapsible state gated by a new localStorage key `partners-list-collapsed`. Pattern: mirror `data-display.tsx:123-134` (`charts-expanded`) but **invert the default** so first-ever visit (no stored value) renders collapsed.

Implementation outline:

```tsx
// Near the top of the component body (after existing useState/useMemo calls):
const [partnersExpanded, setPartnersExpanded] = useState(() => {
  if (typeof window === 'undefined') return false; // SSR: collapsed default
  // On first-ever visit (null), default to collapsed (POL-02).
  // Thereafter: 'false' in storage means collapsed, 'true' means expanded.
  return localStorage.getItem('partners-list-collapsed') === 'false';
});

const togglePartners = useCallback(() => {
  setPartnersExpanded((prev) => {
    const next = !prev;
    // Key semantics: 'false' = NOT collapsed (expanded); 'true' = collapsed.
    localStorage.setItem('partners-list-collapsed', String(!next));
    return next;
  });
}, []);
```

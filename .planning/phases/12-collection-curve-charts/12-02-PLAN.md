```
<div className="w-full space-y-2">
```

**Key anti-patterns to avoid (from research):**
- Do NOT use `type="category"` on XAxis -- must be `type="number"` for proportional spacing
- Do NOT set missing data to 0 -- use undefined (handled by pivot utility)
- Do NOT conditionally render Line components -- use the `hide` prop instead to avoid animation conflicts
- Do NOT use batch names directly as dataKey -- use sanitized keys from pivot utility

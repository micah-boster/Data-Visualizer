Create `src/lib/views/types.ts`:
- `ViewSnapshot` interface with 6 fields:
  - `sorting: SortingState` (from @tanstack/react-table)
  - `columnVisibility: VisibilityState` (from @tanstack/react-table)
  - `columnOrder: string[]`
  - `columnFilters: Record<string, unknown>` (in-column filters only, NOT dimension filters)
  - `dimensionFilters: Record<string, string>` (partner, type, batch from URL params)
  - `columnSizing: ColumnSizingState` (from @tanstack/react-table)
- `SavedView` interface:
  - `id: string` (crypto.randomUUID())
  - `name: string`
  - `snapshot: ViewSnapshot`
  - `createdAt: number` (Date.now())
  - `isDefault?: boolean` (true for starter views)

Create `src/lib/views/schema.ts`:
- Zod schema matching the `SavedView` and `ViewSnapshot` types
- `viewSnapshotSchema` with:
  - `sorting`: z.array(z.object({ id: z.string(), desc: z.boolean() }))
  - `columnVisibility`: z.record(z.string(), z.boolean())
  - `columnOrder`: z.array(z.string())
  - `columnFilters`: z.record(z.string(), z.unknown())
  - `dimensionFilters`: z.record(z.string(), z.string())
  - `columnSizing`: z.record(z.string(), z.number()).optional().default({})
- `savedViewSchema` wrapping the snapshot with id, name, createdAt, isDefault
- Export `savedViewsArraySchema = z.array(savedViewSchema)`
- Note: Use Zod 4 syntax — check `node_modules/zod/` for the correct import path. The project has `"zod": "^4.3.6"` which uses `import { z } from 'zod'` but may have changed import paths in v4.

Create `src/lib/views/storage.ts`:
- `VIEWS_STORAGE_KEY = 'bounce-dv-saved-views'` (distinct from `bounce-dv-columns`)
- `loadSavedViews(): SavedView[]` — reads from localStorage, runs through `savedViewsArraySchema.safeParse()`, returns empty array on failure. SSR-safe (check `typeof window`).
- `persistSavedViews(views: SavedView[]): void` — JSON.stringify and write to localStorage. SSR-safe, try/catch for quota errors.
- Follow the exact same SSR-safe pattern as `src/lib/columns/persistence.ts` (typeof window check, try/catch).

- `useSavedViews()` hook returning:
  - `views: SavedView[]` — current list
  - `saveView(name: string, snapshot: ViewSnapshot): void` — creates new view with crypto.randomUUID(), Date.now(), appends to list
